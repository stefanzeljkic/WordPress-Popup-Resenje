Top and Bottom popup plugin for displaying custom HTML/CSS messages on top or bottom of page. Separate settings for each position.

== Description ==

Simple popup plugin that shows customizable top and/or bottom popups with user-defined HTML content. Responsive design, configurable position, interval, background, padding, and more.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the "Popup Resenje" menu to configure settings.

== Frequently Asked Questions ==

= Can I add my own HTML and CSS? =

Yes. You can add any valid HTML/CSS into the content box for both top and bottom popups.
